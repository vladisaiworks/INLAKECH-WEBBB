# PROJECT_MANIFESTO
## ILAKECH PROJECT
### Contexto Estratégico y Directrices Fundamentales

---

## 1. Cómo debes usar este documento

Este documento define el **marco esencial** de ILAKECH PROJECT.

Debes usarlo como:
- Contexto base antes de generar cualquier contenido.
- Referencia principal para entender el proyecto, su estado y su intención.
- Filtro para evitar interpretaciones erróneas o enfoques incoherentes.

Este documento **no es marketing**, **no es storytelling final** y **no define diseño**.
Es una **orientación estratégica clara**.

Si alguna decisión, texto o solución contradice este documento, **debe revisarse**.

---

## 2. Qué es ILAKECH PROJECT

ILAKECH PROJECT es un **ecosistema regenerativo integral** ubicado en Yucatán,
México.

Es un proyecto que integra:
- Territorio y arquitectura consciente.
- Comunidad y gobernanza estructurada.
- Capital paciente orientado al largo plazo.

No es un desarrollo inmobiliario convencional ni una propuesta espiritual abstracta.
Es un **sistema real, diseñado para ser construido, habitado y sostenido en el tiempo**.

---

## 3. En qué punto se encuentra el proyecto

ILAKECH PROJECT **ya ha comenzado**.

Hechos clave:
- La tierra está asegurada.
- El plan maestro y el diseño arquitectónico están definidos.
- Existe un equipo base real.
- El proyecto entra en su **fase de cimentación**.

No se trata de una idea en fase conceptual, sino de un proyecto que se prepara para su
materialización física.

---

## 4. Objetivo de la landing page

La landing page tiene un objetivo claro y único:

> **Atraer inversores conscientes para acompañar la fase de cimentación del proyecto.**

La landing:
- No vende productos financieros.
- No promete rentabilidades explícitas.
- No genera urgencia artificial.

Su función es:
- Explicar el proyecto con claridad.
- Transmitir confianza, orden y profundidad.
- Permitir que el inversor adecuado se reconozca y decida avanzar solicitando el dossier.

---

## 5. A quién te diriges y cómo debes hablarle

El público objetivo es el **Pionero Visionario**.

Este perfil:
- Busca sentido además de retorno.
- Valora coherencia, estructura y largo plazo.
- Rechaza el marketing agresivo y la especulación.
- Prefiere comprender antes que ser convencido.

El tono debe ser:
- Sereno.
- Adulto.
- Claro.
- Respetuoso.
- Sin hype ni promesas grandilocuentes.

Tu función no es persuadir, sino **permitir una decisión consciente**.

---

## 6. La esencia de ILAKECH

ILAKECH se basa en el principio ancestral:

> **In Lak’ech — “Yo soy otro tú”.**

Este principio no es simbólico, es **operativo**.

Debe reflejarse en:
- La gobernanza (sociocracia).
- La comunicación.
- La forma de presentar el capital.
- La relación entre personas, territorio y decisiones.

Evita cualquier enfoque que genere:
- Imposición.
- Jerarquía innecesaria.
- Fricción artificial.
- Sensación de explotación o urgencia.

ILAKECH no grita.
**Respira.**
